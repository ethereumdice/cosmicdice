How it works

A random number is generated using atmospheric noise. If you guess it correctly, you win! The random numbers are retrieved and winnings are processed automatically by smart contract in the Ethereum blockchain. Using random numbers from atmospheric noise ensures that our numbers are truly random. Smart contracts ensure that payouts are processed instantly. No account needed to play. For more details see the FAQ.

FAQ

How do I play?
No account needed. Simply choose a number and a bet amount. In order to play you need to use a web3 enabled browser to interact with the Ethereum blockchain. The easiest way to enable web3 is by adding the MetaMask extension to your browser. You can also use the Ethereum Mist browser or any other web3 enabled browser.
What is atmospheric noise?
Atmospheric noise is radio noise caused by natural atmospheric processes, primarily lightning discharges in thunderstorms. On a worldwide scale, there are about 40 lightning flashes per second – ≈3.5 million lightning discharges per day. Read more about it on Wikipedia.
What is Ethereum?
Ethereum is a decentralized platform that runs smart contracts: applications that run exactly as programmed without any possibility of downtime, censorship, fraud or third party interference. Read more about it on the Ethereum project website.
How does this actually work?
When you submit your lucky number your browser simultaneously requests a random number from random.org and submits this to the smart contract on the Ethereum blockchain. When the contract receives these data, it requests a new psuedo random number from the Wolfram Alpha computing engine via oraclize.it and calculates another psuedo random number by taking the hash of the timestamp in the Ethereum blockchain. These three numbers are then multiplied together and reduced to an integer by modulo operation. The smart contract then compares your lucky number with this result and makes a payout if they match.
What is a true random number vs a psuedo random number?
Read about it on Wikipedia here: True vs. Psuedo Random Numbers
